# Third-party Dependencies

violetOS relies on the following excellent open-source projects:

## Runtime

* **RPi4 UEFI Firmware** (pftf/RPi4)
  * License: BSD-2-Clause-Patent
  * Used to boot the system on Raspberry Pi 4 hardware.

## Build

* **disk-image-step** (zig-osdev)
  * License: MIT
  * Used to generate the bootable disk images.
